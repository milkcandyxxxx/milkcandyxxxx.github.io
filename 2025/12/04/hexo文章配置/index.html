## 一、文章说明
这是一篇 **Hexo 全字段模板文章**，包含了所有常用的 Front-matter 配置，以及 Markdown 正文的常用写法演示（标题、段落、代码块、公式、流程图等）。

### 1.1 模板使用方法
1. 复制本文到 Hexo 博客的 `source/_posts` 目录下；
2. 修改 Front-matter 中的 `title`、`date`、`categories`、`tags` 等核心字段；
3. 替换正文内容为自己的文章（Markdown 格式）；
4. 执行 `hexo g` 生成静态文件，`hexo d` 部署到远程，或 `hexo s` 本地预览（http://localhost:4000）。

三、关键配置注意事项

  1. Front-matter 格式：必须用 --- 包裹在文章最顶部，字段名冒号后需加空格（如 title: 标题，不能写成 title:标题）；
  2. 必填字段：title 和 date 不可缺失，draft: false 是文章发布的必要条件；
  3. 插件依赖：abbrlink、mermaid、mathjax 等功能需安装对应插件（参考下方命令）；
  4. 主题兼容性：cover、top、copyright 等字段需主题支持，若配置后无效果，可查看主题文档确认是否支持。

必备插件安装命令（终端执行）

# 1. 短链接插件（abbrlink）
npm install hexo-abbrlink --save

# 2. Mermaid 流程图插件
npm install hexo-filter-mermaid-diagrams --save

# 3. 代码块复制按钮插件
npm install hexo-code-block-copy --save

# 4. 阅读时长统计插件
npm install hexo-wordcount --save

四、总结

本文模板包含了 Hexo 文章的所有核心配置和常用功能，新手可直接复用，减少配置成本。如果需要添加特殊功能（如音乐嵌入、视频嵌入），可根据主题/插件需求在 Front-matter 中添加对应字段（如 music: {id: "123456", type: "netease"}，需主题支持网易云音乐嵌入）。

更多 Hexo 配置细节可参考 Hexo 官方文档。


### 模板核心优势：
1. **全字段覆盖**：包含基础配置、分类标签、SEO 优化、访问控制、特殊功能（公式、流程图）等所有常用字段；
2. **直接复用**：修改少量核心字段（标题、分类、正文）即可发布，无需从零配置；
3. **避坑指南**：内置字段说明、插件安装命令、格式注意事项，新手不易出错；
4. **兼容性强**：适配大多数 Hexo 主题（Next、Butterfly、Fluid 等），特殊字段不支持时会自动忽略，不影响文章发布。

### 使用步骤：
1. 复制上述代码，在 `source/_posts` 目录新建文件（如 `hexo-full-template.md`），粘贴内容；
2. 按自己的需求修改 `title`、`categories`、`tags`、`excerpt` 等字段；
3. 替换正文部分为自己的文章内容（Markdown 格式）；
4. 安装需要的插件（如流程图、短链接插件，终端执行模板中的 `npm install` 命令）；
5. 执行 `hexo s` 本地预览，确认无误后 `hexo g -d` 一键生成并部署。
