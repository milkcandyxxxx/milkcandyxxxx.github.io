html#

与其说是语言，倒不如说是一种标记
html负责告诉浏览器如何显示内容，显示什么内容（有点像写作文时的格式）

<html> <!--文章的开始-->
    <head><!--页面的头部标题开始-->
        <title>welcome to milkcandy</title><!--页面的头部标题-->
            <style type="text/css"> 
                body { background-color :#4fb8c8}
            </style> <!--CSS样式，背景颜色-->
    </head>
    <img src="https://i.postimg.cc/NMsqpxvM/111214302-p0.jpg"> <!--插入的图片-->
    <body><!--主体开始-->        
        <H1><!--标题1-->
            <em><!--斜体-->
                这里是标题1，采用了斜体<br>
                    <q><!--引用（短）-->
                        斜体加引用
                    </q><br>
            </em>
            <q>只是引用</q>
        </H1>
        <p><!--内容-->
            这里是标题1的内容<br>
            <code>显示代码，bie kan le wo bu hui!</code><br><!--显示代码-->
            <time>时间文本</time><br><!--时间-->
            <strong>强调文本</strong><br><!--strong强调文本(加粗)-->
            <pre>按照原文本格式输出</pre><br><!--指定文本格式-->
            <a href="../about/about.html">这里是跳转页面</a><br><!--指向页面-->
            <a href="https://i.postimg.cc/NMsqpxvM/111214302-p0.jpg">跳转图像</a><br><!--指向图像-->
        </p>
        <blockquote>这里是长引用~~~寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱<br><!--换行-->
            寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱寻酱</blockquote><!--引用（长）-->
        <ol><!--排序（有序）--><!--ul排序（无序）-->
            这里是排序
            <li>xunjiang</li><!--包裹内容存放于元素ol-->
            <li><em>laogong!</em>qianxun</li>
            <li>hequanshawu</li>

        </ol>       
    </body>
</html>

其他#

再往后就是一些嵌套，路径等问题，总的来说就是要细心，还有在早些时候就做好内容的整理
最后一点就是对于web的部分讲解，大都是理论部分：域名，FTP,URL,HTTP等
域名：用于定位网站就是我的www.milkcandy.shop
FTP:网络协议，这里了解是为了上传自己的网页到云服务器
cd:跳转到指定目录/put：上传/mkdir：新建目录/get：获取文件
HTTP:超文本传输协议（HyperText Transfer Protocol）,一种简单的请求和响应协议
绝对路径：..//,类似如此部属最少的，从一个文件到另一个
默认页面：像网页发送请求后默认响应的文件
URL:资源地址，奶糖博客中的图片均是填写的url，打开时候再访问。
