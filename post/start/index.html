前言

总的来说，目前还是要定一下自己的学习目标吧。

why？

先解释一下为什么我身为高三畜却有这些闲情逸致来学习其他的事情，因为奶糖是已经要参加一种名为“单招”的东西，不参加高考了。不过这个所谓的“单招”只是高职院校的考试而已。奶糖的学习并不好，用尽了全力才考上了个不算差的高中，但也是随着两年的生活过去，因为偏科，以及的的确确奶糖的学习能力支撑不了7门学科的学习，如果还要硬着头皮选择的高考的话，奶糖很可能（100%是）是炮灰啦（；´д｀）ゞ。因为单招相对简单，且科目较少，所以奶糖走这样一条路的话，反而能考上一些不错的高职（高考中比一些本科分都高哩）。
所以，因为奶糖的选择奶糖的学习压力一下子清了许多。所以奶糖才有时间去维护自己的爱好。

奶糖很早就喜欢这一类的东西，从scratch到micro:bit，再之是python，这些学习路上有幸遇见的都让奶糖乐此不疲。不过真正起步的还是疫情期间的yunzai（qq机器人，奶糖用的版本链接找不到了），最一开始没有成熟的教程及一键部署，奶糖一步一个脚印的尝试。再后来第一次接触博客是timochan的（目前还未取得认可就不放链接啦），大佬的博客真的好看(❤ ω ❤)，不过因为线下的开始，学习压力没有给我太多的时间，也是时至今日，奶糖才开始为之行动。

起步

由于奶糖也不知道到了大学里自己会学些什么，奶糖的理想应该是当一个网络安全工程师，不过高职院校不开设相关专业/(ㄒoㄒ)/~~（不过奶糖也不会放弃哒！！！自学也是学嘛）
但是奶糖的起步没有想是理想方面的。还记得我博客的主题是用的youranreus大佬的开源主题，所以奶糖也想写出一个属于自己的主题（其实看官方文档以及与开源主题的比对，奶糖大概能知道该是怎样的形式，我想如果不停的仿照，修改，重试，应该能拼拼凑凑的弄出来一个，但是这样奶糖肯定理解不了其中真实的写法，含意），奶糖想从htlm学起，再往后到php从静态的逐步往后嘛d=====(￣▽￣*)b，奶糖也要自己写出好看的css！！！

如何学习

奶糖也不太清楚自己该怎么去系统的学习这些东西，以前奶糖对于此只是爱好而已，现在到了真正去学习的时候反倒有了些许迷茫(。_。)。奶糖在网上也看了许多大佬分享的经验，有了些许眉目吧。上文所说的奶糖想从前端学起，先就是网页入门的html和css啦！！！（加油(。・∀・)ノ）
